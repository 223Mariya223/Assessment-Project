This repository contains a Selenium Java automation testing project developed as part of an assessment for n11. The project demonstrates web UI test automation using Selenium WebDriver, Java, Maven, and TestNG,
following standard automation framework practices.

The framework is structured to be easy to understand, maintain, and extend, making it suitable for real-world test automation scenarios.



Programming Language: Java

Automation Tool: Selenium WebDriver

Test Framework: TestNG

Build Tool: Maven

IDE: Eclipse 

Version Control: Git & GitHub


1. src/test/java # Test classes and automation scripts
2. Screenshots # Screenshots captured during test execution
3.test-output # TestNG execution reports
4.pom.xml # Maven dependencies and configuration
5.gitignore # Git ignored files
6.project # Eclipse project configuration
7.classpath # Eclipse classpath configuration
8. README.md # Project documentation
